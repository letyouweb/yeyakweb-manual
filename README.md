PuppyHotel 관리자 대시보드 메뉴얼﻿
이 메뉴얼은 PuppyHotel 관리자 대시보드를 처음 사용하는 분들을 위해 쉽게 설명한 안내서입니다. 초등학생도 이해할 수 있도록 하나씩 천천히 설명하니, 차근차근 따라 해 보세요.

1. 캘린더 설정﻿
캘린더가 하는 일
손님들이 언제든지(24시간) 미용·호텔·데이케어 서비스를 예약할 수 있게 해 주는 AI 상담 챗봇의 예약표입니다.
우리가 영업하는 시간과 요일, 하루에 받을 수 있는 예약 수를 정하면 챗봇이 그 시간에 맞춰 예약을 받습니다.

사용하는 방법
캘린더 설정 메뉴를 열면, 영업시간과 예약 수를 설정할 수 있는 칸이 나옵니다.
시작 시간과 종료 시간을 선택하세요. 예를 들어 오전 9시부터 저녁 8시까지 영업한다면 시작은 09:00, 종료는 20:00으로 맞춥니다.
일일 최대 예약 칸에 숫자를 적어 하루에 최대 몇 건의 예약을 받을지 정합니다. 숫자가 클수록 많은 손님을 받을 수 있지만, 너무 많으면 서비스가 힘들 수 있습니다.
모든 설정이 끝났다면 “캘린더 설정 완료하기” 버튼을 누릅니다. 그러면 설정이 저장되고 챗봇이 그에 맞춰 자동으로 예약을 받습니다.
💡 TIP: 캘린더를 켜 두면 손님이 밤늦게나 새벽에도 예약 문의를 할 수 있습니다. 필요할 때만 켜두세요.

2. 요일별 미용예약 설정﻿
무엇을 설정하나요?
미용 서비스는 요일마다 영업 여부와 예약 가능한 시간을 다르게 설정할 수 있습니다.
예를 들어, 월~금요일은 오전 9시부터 오후 5시까지 열고, 토요일은 오전 시간만 열 수 있습니다.
요일마다 하루에 받을 수 있는 미용 예약 건수도 정할 수 있습니다.

설정하는 방법
요일별 미용예약 설정 메뉴를 누르면 월요일부터 일요일까지 표가 나옵니다.
각 요일의 “운영” 스위치를 켜면 그날 미용 서비스를 합니다. 스위치를 끄면 그날은 미용 서비스를 하지 않습니다.
운영으로 설정한 요일은 최대 예약 수를 지정할 수 있습니다. 예를 들어 월요일에 8로 설정하면, 하루에 8마리까지만 예약을 받습니다.
예약 가능 시간 줄에서 원하는 시간을 추가할 수 있습니다. 09:00, 10:00 등 버튼을 눌러 시간을 입력하면 추가되고, 옆의 × 버튼을 누르면 제거됩니다.
모든 요일의 설정이 끝났다면 오른쪽 위의 “설정 저장” 버튼을 눌러 저장합니다.
📌 주의: 시간을 지정하지 않으면 손님이 그 요일에 아무 시간이나 선택할 수 있습니다. 꼭 운영하는 시간을 입력해 주세요.

3. 예약 관리﻿
여기서 하는 일
지금까지 들어온 모든 예약을 한 눈에 볼 수 있는 곳입니다.
예약을 확정, 대기, 완료, 취소와 같은 상태로 바꾸거나 삭제할 수 있습니다.
여러 예약을 한 번에 선택해서 삭제할 수도 있습니다.

화면 설명
열 이름	뜻	주요 기능
반려동물	강아지 또는 고양이 이름	해당 예약의 대상입니다.
보호자	주인 이름	연락 시 참고합니다.
서비스	호텔/미용/데이케어 구분	아이콘으로 서비스 종류를 표시합니다.
날짜/시간	예약 날짜와 시간	미용과 데이케어는 시간, 호텔은 체크인 날짜가 표시됩니다.
상태	확정/대기/완료/취소	색깔 배지로 표시되며 클릭해서 상태를 바꿀 수 있습니다.
연락처	전화번호	주인과 연락할 때 클릭하면 전화를 걸 수 있습니다.
관리	확정/완료/취소 버튼	예약 상태를 바꾸는 버튼입니다.
선택	체크박스	선택 후 상단의 삭제 버튼을 누르면 여러 예약을 한꺼번에 삭제할 수 있습니다.
사용하는 방법
예약 목록에서 원하는 예약의 상태를 변경하려면 확정, 대기, 완료, 취소 중 하나의 버튼을 누릅니다. 예를 들어, 미용 예약이 확정되면 “확정” 버튼을 누릅니다.
예약이 대기 상태일 때 노란색 배지를 클릭하면, 그 예약이 속한 서비스(미용·호텔·데이케어) 탭으로 이동하여 상세한 일정을 확인할 수 있습니다.
여러 예약을 지우고 싶다면 각 예약 왼쪽의 체크박스를 클릭해 선택하고, 맨 위 삭제 버튼을 누르세요. 삭제 전에는 확인 메시지가 뜹니다.
⚠️ 삭제 주의: 삭제하면 예약이 완전히 없어집니다. 잘못 지우지 않도록 확인 후 삭제하세요.

4. 미용예약현황﻿
무엇이 보이나요?
오늘 미용, 대기 중, 완료, 총 예약 등 통계 카드가 있어 이번 주와 오늘 미용 예약이 얼마나 되는지 한눈에 알 수 있습니다.
미용 예약 달력이 있어 날짜별 예약 상태를 색깔로 표시합니다.

회색: 예약 없음

초록색: 여유 (1~2건)

노란색: 보통 (3~4건)

빨간색: 포화 (5건 이상)

달력에서 날짜를 클릭하면 그 날의 예약 목록이 아래에 나타나고, 예약 추가 버튼으로 새 예약을 직접 입력할 수 있습니다.
예약 목록 옆의 삭제 버튼을 눌러 개별 예약을 지울 수 있습니다.

새 미용 예약 추가하기
달력에서 원하는 날짜를 클릭합니다. 날짜를 누르면 아래에 “미용 예약 현황” 박스가 열립니다.
박스 안에서 예약 추가 버튼을 누르면 입력 폼이 나타납니다.
반려동물 이름, 보호자 이름, 시간, 연락처, 미용 스타일을 적습니다. 모두 적지 않아도 기본 값이 채워집니다.
추가 버튼을 누르면 예약이 저장되고 리스트에 나타납니다.

예약 삭제하기
예약 옆에 있는 삭제 버튼을 누르면 즉시 삭제됩니다. 삭제하면 취소할 수 없으니 꼭 확인하세요.

5. 호텔﻿
호텔 메뉴 살펴보기
오늘 체크인, 현재 투숙, 오늘 체크아웃, 총 예약 카드에서 호텔 예약 현황을 빠르게 확인합니다.
호텔 예약 달력을 통해 날짜별 객실 예약 상황을 색상으로 구분해 보여줍니다.

초록색: 여유 (1~3건)

노란색: 보통 (4~6건)

빨간색: 만실 (7건 이상)

달력에서 날짜를 클릭하면 아래에 그 날 체크인하는 손님 목록이 표시되고, 예약 추가 버튼을 통해 새 예약을 입력할 수 있습니다.
체크인/체크아웃 날짜, 룸 타입, 보호자 이름, 연락처 등을 입력하여 새 예약을 만들 수 있고, 예약 옆의 삭제 버튼으로 삭제도 할 수 있습니다.

새 호텔 예약 추가하기
달력에서 원하는 체크인 날짜를 클릭합니다. 날짜 선택창 아래에 예약 목록 박스가 뜹니다.
예약 추가 버튼을 누르면 입력 폼이 보입니다.
반려동물 이름, 보호자 이름, 체크아웃 날짜, 룸 타입, 연락처를 적습니다.
추가를 누르면 예약이 저장되고 해당 날짜에 표시됩니다.
🛏️ 참고: 룸 타입은 적지 않으면 ‘일반룸’으로 자동 지정됩니다.

6. 데이케어﻿
데이케어 메뉴 살펴보기
오늘 등원, 현재 돌봄, 완료, 총 예약 카드로 데이케어 상황을 한눈에 확인합니다.
데이케어 예약 달력은 평일만 색상으로 표시합니다. 주말은 회색(휴무)으로 표시되어 클릭할 수 없습니다.

초록색: 여유 (1~2건)

노란색: 보통 (3~4건)

빨간색: 포화 (5건 이상)

달력에서 평일 날짜를 클릭하면 그 날의 데이케어 예약 목록이 나타납니다. 예약 추가 버튼으로 시간, 반려동물 이름, 보호자 이름, 연락처를 입력해 새 예약을 넣을 수 있습니다.
예약 목록 옆의 삭제 버튼을 통해 예약을 삭제할 수 있습니다.

새 데이케어 예약 추가하기
달력에서 원하는 평일 날짜를 클릭합니다. 날짜를 누르면 아래에 “데이케어 예약 현황” 박스가 열립니다.
예약 추가 버튼을 눌러 입력 폼을 엽니다.
반려동물 이름, 보호자 이름, 시간을 선택하고 연락처를 적습니다.
추가를 누르면 예약이 저장되고 리스트에 추가됩니다.

🚫 주의: 데이케어는 주말(토요일, 일요일)에는 운영하지 않습니다. 주말은 달력에서 회색으로 표시되어 선택할 수 없습니다.

7. 계정 관리﻿
무엇을 할 수 있나요?
관리자 계정의 아이디, 이메일, 보안 질문을 변경하거나 확인할 수 있습니다.
비밀번호를 새로운 것으로 변경할 수 있습니다.
정보 수정 후 저장 버튼을 눌러야 변경 사항이 적용됩니다.

사용하는 방법
계정 관리 메뉴를 클릭합니다.
수정 버튼을 누르면 입력칸이 활성화됩니다. 수정이 필요 없는 칸은 그대로 두어도 됩니다.
아이디, 이메일, 보안 질문·답변을 입력한 후 저장 버튼을 누르면 변경 내용이 저장됩니다.
비밀번호를 바꾸려면 새로운 비밀번호와 확인용 비밀번호를 입력하고 저장을 누르세요. 비밀번호는 최소 8자리 이상이어야 합니다.

🔒 보안 팁: 비밀번호는 다른 사이트와 다르게 설정하고, 정기적으로 변경하는 것이 좋습니다.
